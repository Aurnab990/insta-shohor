How does javascript work?

Javascript কোড ব্রাউজার দ্বারা লোড হয়। ব্রাউজার scripts read করে ফাইল থেকে এবং execute করা শুরু করে৷ এটি করে থাকে Javascript engine৷ যেহেতু javascript একটি interpreted language, ফলে এর compiler দরকার হয় না৷ এটি directly read করে execute করে কোড।

What does js event loop do?

আমরা জানি javascript একটি single threatened language, এটি শুধুমাত্র একটি কমেন্ট রান করতে পারে প্রতিবার৷ coment গুলো asynchronously হয়। এটার জন্য responsible Event Loop৷ Call stack থেকে সম্পূর্ণ execution শেষ হয়ে তারপর Event Loop এ Pop করে statement গুলো এবং তারপর push করে Event Queue তে ৷ এভাবে Event Loop কাজ করে।

Local Storage vs Session Storage

The main difference between them is , if i store a data in session storage , the data will be cleared if i back to previous page.
On the other hand, if i store data in Local storage , the data will be permanantly saved for future.And I can see the data if i refresh the page again and again.
